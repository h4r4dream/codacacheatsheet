The Cheat Sheet

HTML

HTML Elements

List item elements

  • <ul> - Defines an unordered list.
  • <ol> - Defines an ordered list.
  • <li> - Defines a list item.

Text content elements

  • <p> - Defines a paragraph.
  • <h1> to <h6> - Defines HTML headings.
  • <blockquote> - Defines a section that is quoted from another source.

Video/Audio elements

  • <video> - Defines a video or movie.
  • <source> - Specifies multiple media resources for elements like <video> and <audio>.
  • <controls> - Adds video controls, like play, pause, and volume.

Emphasis elements

  • <strong> - Defines important text.
  • <em> - Defines emphasized text.
  • <mark> - Defines marked or highlighted text.

Form elements

  • <form> - Defines an HTML form for user input.
  • <input> - Defines an input control.
  • <label> - Defines a label for an <input> element.
  • <button> - Defines a clickable button.
  • <select> - Defines a drop-down list.
  • <option> - Defines an option in a drop-down list.

Link elements

  • <a> - Defines a hyperlink.
  • <link> - Defines the relationship between a document and an external resource (most used to link to stylesheets).

Image elements

  • <img> - Embeds an image in the document.
  • <src> - specifies the source of the image
  • <alt> - provides alternative text for the image

Attributes

  • class - Specifies one or more class names for an element.
  • id - Specifies a unique id for an element.
  • style - Specifies inline CSS styles for an element.

HTML Structure

HTML Boilerplate

  • <!DOCTYPE html> - Defines the document type and version of HTML.
  • <html> - The root element of an HTML page.
  • <head> - Contains meta-information about the document.
  • <body> - Contains the content of the document.

Meta Tags

  • <meta> - Defines metadata about an HTML document.
  • <title> - Specifies the title of the document (shown in the browser's title bar or tab).
  • <description> - Provides a brief description of the page (used by search engines).
  • <viewport> - Controls the layout on mobile browsers.

HTML Comments

  • <!-- Comment --> - Defines a comment in HTML (not displayed in the browser).

HTML Entities

  • & - Represents the ampersand (&) character.
  • < - Represents the less-than (<) character.
  • > - Represents the greater-than (>) character.
  • " - Represents the double-quote (") character.
  • ' - Represents the single-quote (') character.

Anchor Elements

  • <a> - Defines a hyperlink.
  • <link> - Defines the relationship between a document and an external resource (most used to link to stylesheets).

HTML Semantic Elements

  • <header> - Represents introductory content or a group of navigational links.
  • <nav> - Defines a set of navigation links.
  • <main> - Represents the main content of the document.
  • <article> - Represents a self-contained composition in a document.
  • <section> - Defines a section in a document.
  • <aside> - Represents content that is tangentially related to the content around it.
  • <footer> - Represents the footer for its nearest sectioning content or sectioning root element.

HTML Tables

  • <table> - Defines a table.
  • <tbody> - Groups the body content in a table.
  • <tr> - Defines a table row.
  • <td> - Defines a table data element.
  • <th> - Defines a table header cell.
  • <caption> - Defines a table caption.
  • <colspan> - Defines the number of columns a cell should span.
  • <rowspan> - Defines the number of rows a cell should span.
  • <thead> - Groups the header content in a table.
  • <tfoot> - Groups the footer content in a table.

CSS

CSS Selectors

  • * - Universal selector (selects all elements).
  • element - Type selector (selects all elements of a given type).
  • .class - Class selector (selects all elements with a given class).
  • #id - ID selector (selects a single element with a given ID).
  • [attribute] - Attribute selector (selects elements with a specific attribute).
  • :hover - Pseudo-class (selects elements when hovered over).
  • ::before - Pseudo-element (selects content before an element).

Selector specificity

Selector specificity determines which CSS rule is applied when multiple rules match the same element. It is calculated based on the types of selectors used in a rule:

  • Inline styles (added directly to an element) have the highest specificity.
  • ID selectors have a higher specificity than class selectors.
  • Class selectors have a higher specificity than type selectors.
  • Universal selectors have the lowest specificity.

CSS link element

The <link> element is used to link external resources, such as stylesheets, to an HTML document. It is placed within the <head> section of the document.

  • rel - Specifies the relationship between the current document and the linked resource (e.g., stylesheet).
  • href - Specifies the URL of the linked resource.
  • type - Specifies the MIME type of the linked resource (e.g., text/css).

CSS Box Model

The CSS box model describes the rectangular boxes generated for elements in the document tree and consists of the following areas:

  • Content - The innermost area where text and images appear.
  • Padding - The space between the content and the border, inside the element.
  • Border - A line surrounding the padding (if any) and content.
  • Margin - The outermost space around the element, separating it from other elements.

Box Model Properties

The following CSS properties are used to control the box model:

  • width - Sets the width of the content area.
  • height - Sets the height of the content area.
  • padding - Sets the padding space around the content.
  • border - Sets the border around the padding and content.
  • margin - Sets the margin space outside the border.

Box Model Sizing

Box model sizing can be affected by the following CSS properties:

  • box-sizing - Defines how the width and height of an element are calculated.
  • overflow - Controls what happens to content that overflows the element's box.

CSS margin collapse

Margin collapse is a behavior in CSS where the vertical margins of block-level elements can combine (or "collapse") into a single margin. This can occur in the following situations:

  • When two block-level elements are adjacent to each other (e.g., one above the other).
  • When a block-level element is the first or last child of its parent.
  • When a block-level element has no border, padding, or content.

CSS auto keyword

The auto keyword in CSS is used to enable the browser to calculate a value automatically. It can be applied to various CSS properties, including:

  • width and height - When set to auto, the browser determines the size of the element based on its content and surrounding elements.
  • margin - Setting the margin to auto can center an element horizontally within its container.
  • position - When used with position: absolute, auto can allow the element to be positioned based on its nearest positioned ancestor.

CSS overflow property

The overflow property in CSS controls how content that overflows an element's box is handled. It can take the following values:

  • visible - The content is not clipped and may overflow the element's box.
  • hidden - The content is clipped, and no scrollbars are provided.
  • scroll - The content is clipped, but scrollbars are provided to view the overflowing content.
  • auto - The browser determines whether to provide scrollbars based on the content size.

CSS visibility property

The visibility property in CSS controls the visibility of an element without affecting its layout. It can take the following values:

  • visible - The element is visible.
  • hidden - The element is not visible, and it does not take up any space in the layout.
  • collapse - For table rows and columns, this value removes the element from the layout (similar to display: none).

CSS Display and positioning

CSS display and positioning properties control how elements are displayed and positioned within the document. Here are some key properties:

CSS Z-index property

The z-index property in CSS controls the stacking order of elements that overlap. It can take the following values:

  • auto - The default value, which means the element is stacked according to the order in the HTML.
  • number - A positive or negative integer that determines the element's stack level. Higher values are stacked on top of lower values.

Fixed CSS Positioning

Fixed positioning is a CSS positioning method that allows you to position an element relative to the viewport, meaning it stays in the same place even when the page is scrolled. Here are some key points about fixed positioning:

  • To use fixed positioning, set the position property to fixed.
  • Fixed elements are removed from the normal document flow, so they do not affect the position of other elements.
  • You can use the top, right, bottom, and left properties to specify the position of a fixed element.

CSS display property

The display property in CSS determines how an element is displayed on the page. It can take the following values:

  • block - The element is displayed as a block-level element, taking up the full width available.
  • inline - The element is displayed as an inline element, taking up only as much width as necessary.
  • inline-block - The element is displayed as an inline-level block container.
  • none - The element is not displayed at all (it is removed from the document flow).

CSS position: absolute

The position: absolute property in CSS allows you to position an element relative to its nearest positioned ancestor (i.e., an ancestor with a position other than static). Here are some key points about absolute positioning:

  • To use absolute positioning, set the position property to absolute.
  • Absolute elements are removed from the normal document flow, so they do not affect the position of other elements.
  • You can use the top, right, bottom, and left properties to specify the position of an absolute element.

CSS position: relative

The position: relative property in CSS allows you to position an element relative to its normal position in the document flow. Here are some key points about relative positioning:

  • To use relative positioning, set the position property to relative.
  • Relative elements are not removed from the normal document flow, so they still affect the position of other elements.
  • You can use the top, right, bottom, and left properties to specify the position of a relative element.

CSS float property

The float property in CSS is used for positioning and formatting content, allowing text and inline elements to wrap around a floated element. Here are some key points about the float property:

  • To use the float property, set it to left or right to float an element to the left or right of its container.
  • Floated elements are removed from the normal document flow, so they do not affect the position of other elements.
  • To contain floated elements within a parent container, you can use the clearfix technique or set the parent's overflow property to auto or hidden.

CSS clear property

The clear property in CSS is used to control the behavior of floating elements. It specifies whether an element can be next to floating elements or must be moved below them. Here are some key points about the clear property:

  • To use the clear property, set it to left, right, both, or none.
  • The left value prevents floating elements on the left side from affecting the element.
  • The right value prevents floating elements on the right side from affecting the element.
  • The both value prevents floating elements on both sides from affecting the element.
  • The none value allows the element to be next to floating elements.
